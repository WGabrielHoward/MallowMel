# MallowMel 

A cute 2D platformer built in Unity where you control **Mel the Marshmallow** – a bouncy, squishy character navigating sweet-themed levels with fun platforming, power-ups, and scoring!

### Gameplay & Controls
- **Objective**: Bounce, run, and roll through colorful levels, collect sweets for points, grab power-ups, and reach the goal.
- **Desktop Controls**:
  - **A / D** or **Left / Right Arrows** — Move left/right
  - **Space** or **W / Up Arrow** — Jump (with extra-bouncy marshmallow physics!)
- **Mobile Controls**:
  - On-screen touch joystick for movement and jump
- Power-ups add fun visual effects and temporary immunity to hazards!

### Features
- Custom bouncy marshmallow character controller with satisfying squishy physics
- Sweet-themed 2D platforming levels with obstacles and collectibles
- Power-up system and scoring mechanics
- Mobile-friendly touch controls for WebGL
- Custom shaders for stylized visual effects 
- Clean C# architecture and ongoing iterative development

### Tech Stack
- **Engine**: Unity 6
- **Language**: C# (game logic), ShaderLab/HLSL (custom visuals)
- **Key Concepts Demonstrated**:
  - 2D physics and responsive character controllers
  - Mobile WebGL touch input support
  - Custom shaders for stylized effects
  - Power-up and scoring systems
  - Browser deployment

### How to Run Locally
1. Clone the repo: 'git clone https://github.com/WGabrielHoward/MallowMel.git'
2. Open the project in Unity Hub (Unity 6 recommended)
3. Load the main scene ('Assets/Scenes/Scene_MM')
4. Press Play! (Works great in Editor; mobile controls visible in wider windows)
